## Stages of development of OR
- Observe the problem equipment 
- Analyse and define the problem
- Develop a model 
- Select appropiate data input 
- Provide a solution and test 
- Implement the solution

- Finance 
- Manufacturing 
- Marketing 
- Human Resource 
- Purchasing
